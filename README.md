# ニュース速報24 — ニュースサイトのテンプレート（静的）

**静的**なニュースサイトのテンプレート（HTML / CSS / 素の JS）。レイアウトは
一般的なニュースサイトを参考にしています。ブランド名と内容は**自分のもの**を
使ってください — 実在する報道機関のロゴ・名称・記事は含まれていません。

## 構成

```
.
├── index.html       # トップ: ヒーロー + 4つのカテゴリーブロック + サイドバー
├── category.html    # カテゴリーページ（URL の ?cat=... を読み取る）
├── article.html     # 記事詳細ページ（URL の ?id=... を読み取る）
├── css/style.css    # すべてのスタイル（デザイントークンは :root に集約）
└── js/
    ├── articles.js  # 記事データ（デモ14本）— 記事の追加・編集はここ
    └── main.js      # 記事/カテゴリー/サイドバーの描画 + モバイルメニュー + タブ
```

## 記事の追加・編集

`js/articles.js` を開き、`ARTICLES` 配列にオブジェクトを1つ追加します。

```js
{
  id: "kiji-id",              // article.html?id=kiji-id のリンクに使用
  cat: "経済", catSlug: "keizai",
  title: "記事のタイトル",
  sapo: "短い要約。",
  author: "著者名", date: "2026/06/16 09:00",
  views: "1,234", order: 141,  // order が大きいほど新しい
  img: "seed-image",           // picsum 用の seed（実画像に差し替え可）
  tags: ["タグ1", "タグ2"],
  body: ["<p>段落1</p>", "<h2>小見出し</h2>", "<p>段落2</p>"]
}
```

カテゴリー、「よく読まれている / 最新」、「関連記事」はデータに応じて
自動で更新されます。

## 動作確認

`index.html` をブラウザで直接開くか、静的サーバーを起動します。

```bash
# Python
python3 -m http.server 8000

# または Node
npx serve .
```

その後 http://localhost:8000 を開きます。

## カスタマイズ

- **ブランド名 / カラーの変更:** `css/style.css` の `:root` 内の変数
  （`--color-accent` など）と、各 HTML 内の「ニュース速報24」を編集します。
- **画像:** プレースホルダーに `picsum.photos`（要インターネット）を使用中。
  `assets/` に実画像を置き、`src` を差し替えてください。
- **内容:** タイトル・要約・本文を自分の内容に置き換えてください。

## デプロイ

静的サイトなので **Cloudflare Pages**、Netlify、Vercel などにそのまま
デプロイできます。ルートフォルダを指定するだけで、ビルドは不要です。

## 著作権に関する注意

これはオリジナルのテンプレートです。著作権・商標の侵害を避けるため、実在する
報道機関のロゴ・商標・記事・画像は**コピーしないでください**。
